结论分析与体会：

通过本次实验，我首先熟悉了许多环境的配置，其中包括Node.js，在刚开始配置这个环境的时候，在输入 npm install hexo-cli -g时，系统总是会报错，在查阅了相关资料后，我了解到，原来时之前用的淘宝的镜像的认证到期了，所以不能再用，最终换了npm config set registry https://registry.npmmirror.com这条命令后，就能够成功运行了；其次，我还学会了如何在Github上进行部署，掌握了对应的原理和方法，对于如何在Github上搭建自己的博客有了全新的了解和认识，对于以后的云计算学习打下了一定的基础。
